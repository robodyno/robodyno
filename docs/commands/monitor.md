
# robodyno monitor

Monitor all devices on the CAN bus.

## Usage

```
Usage: robodyno monitor [OPTIONS]
```

## Options
* `group`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--group
-g`

  Group messages by CAN ID.


* `help`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--help`

  Show this message and exit.
