
# motor clear-errors

Clear errors on motors.

## Usage

```
Usage: robodyno motor clear-errors [OPTIONS]
```

## Options
* `help`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--help`

  Show this message and exit.
